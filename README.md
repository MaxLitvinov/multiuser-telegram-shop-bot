# 🎲 Borent — Telegram-бот для аренды настольных игр

Telegram-бот для проката настольных игр с корзиной, оплатой через QIWI P2P и админ-панелью.

---

## ⚙️ Функционал

### Для пользователей
- 📋 Просмотр игр в наличии с фото, ценой и залогом
- 🗑 Корзина — добавление/удаление игр, аренда на день или неделю
- 🚗 Выбор способа получения — доставка или самовывоз
- 💳 Оплата банковской картой через QIWI P2P или наличными
- ✉️ Предложить новую игру
- 🖌 Задать вопрос продавцу

### Для администратора
- ➕ Добавить новую игру (название, цены, залог, фото)
- 💥 Удалить игру из базы
- 📪 Вернуть игру в наличие
- 📬 Получение уведомлений о новых заказах и вопросах

## 🚀 Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone <ссылка на репозиторий>
cd telegram_bot
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Заполнить конфиг в `create_bot.py`
```python
TOKEN = "токен от @BotFather"
ADMIN_CHAT_ID = "ID чата для получения заказов"
```

### 4. Заполнить QIWI ключ в `bot.py`
```python
QIWI_KEY = "ваш ключ QIWI P2P"
```

### 5. Запустить
```bash
python main.py
```

## 🗃 База данных

SQLite. Файл: `db/database.db`

Таблица `game_checker` — игры:

| Поле | Тип | Описание |
|---|---|---|
| game_name | TEXT | Название игры |
| available | INT | 1 — в наличии, 0 — недоступна |
| price_day | TEXT | Цена за день |
| price_week | TEXT | Цена за неделю |
| deposit | TEXT | Сумма залога |
| photo | TEXT | file_id фото |

Таблица `users` — пользователи и заказы:

| Поле | Тип | Описание |
|---|---|---|
| username_tg | TEXT | Username в Telegram |
| ordered | INT | Есть ли активный заказ |
| order_text | TEXT | Список игр в заказе |
| rent_price | INT | Сумма аренды |
| order_price | INT | Общая сумма |
| user_adress | TEXT | Адрес доставки |
| delivery | INT | 1 — доставка, 0 — самовывоз |
| pay_method | TEXT | Способ оплаты |
| bill_id | TEXT | ID счёта QIWI |
| suggestion | TEXT | Предложение пользователя |
